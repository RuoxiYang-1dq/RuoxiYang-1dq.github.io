这是最核心、最常用的调用方式。前提是你已经安装了`openai`库（如果没装，先执行`pip install openai`）。

python
from openai import OpenAI

    api_key="你的千聚API Key在这里", # 从千聚官网获取

    ],
    stream=False # 流式输出

**核心点说明：**
-   `base_url`必须是`https://www.qianjuai.com/v1`，这是你告别网络错误的起点。
-   `api_key`从千聚后台生成，注意保管。
-   `model`参数可以换成任何千聚支持的模型名，比如`gpt-4o-mini`或`claude-sonnet-4-20250514`。

### 2. 流式对话调用（Streaming）

流式输出让你能实时看到AI打字的效果，体验更好。只需要把上面的`stream`改为`True`，并循环处理响应：

python
from openai import OpenAI

client = OpenAI(
    api_key="你的千聚API Key在这里",
    base_url="https://www.qianjuai.com/v1"
)

stream = client.chat.completions.create(
    model="gpt-4-turbo",
    messages=[{"role": "user", "content": "给我讲一个关于AI的短故事。"}],
    stream=True # 启用流式输出
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="") # 实时打印


**优势：** 流式输出在国内直连下依然流畅，没有卡顿，完美解决了过去需要梯子才能实现实时交互的痛点。

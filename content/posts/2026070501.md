### 第二步：写一段原生调用代码

这是一个最直接的HTTP请求示例，没有用任何第三方封装的AI库，能让你清楚看到数据流。

java
import okhttp3.*;
import com.alibaba.fastjson.JSON;
import com.alibaba.fastjson.JSONObject;
import com.alibaba.fastjson.JSONArray;

public class GeminiTest {
    public static void main(String[] args) {
        String apiKey = "sk-你的千聚Key";  // 从千聚后台获取
        String url = "https://www.qianjuai.com/v1/chat/completions";  // 关键：改这里

        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "用Java写一个快速排序算法，带注释，用中文解释");

        JSONArray messages = new JSONArray();
        messages.add(message);

        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "gemini-2.0-flash-exp");  // 指定模型
        requestBody.put("messages", messages);
        requestBody.put("max_tokens", 2048);

        Request request = new Request.Builder()
                .url(url)
                .addHeader("Authorization", "Bearer " + apiKey)
                .addHeader("Content-Type", "application/json")
                .post(RequestBody.create(
                        MediaType.parse("application/json"),
                        requestBody.toJSONString()))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful() && response.body() != null) {
                String responseBody = response.body().string();
                JSONObject jsonResponse = JSON.parseObject(responseBody);
                String content = jsonResponse.getJSONArray("choices")
                        .getJSONObject(0)
                        .getJSONObject("message")
                        .getString("content");
                System.out.println("Gemini 回答：\n" + content);
            } else {
                System.out.println("API 请求失败: " + response.code() + " - " + response.message());
                if (response.body() != null) {
                    System.out.println("错误详情: " + response.body().string());
                }
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}


你有没有发现？这个调用的结构和OpenAI完全一样。你之前写的调用GPT-4的method，改个参数就能无缝切到Gemini。

### 第三步：用第三方库更简单

上面是原生调用。如果你的项目用了 `openai-java` 这类第三方库，更简单：

java
import com.theokanning.openai.service.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;

public class GeminiWithLib {
    public static void main(String[] args) {
        // 就这么一行，改base_url
        OpenAiService service = new OpenAiService("sk-你的千聚Key", Duration.ofSeconds(60));

        String apiUrl = "https://www.qianjuai.com/v1";
        System.setProperty("openai.base.url", apiUrl);

        // 其他请求代码完全不用改
        List<ChatMessage> messages = new ArrayList<>();
        messages.add(new ChatMessage("user", "你好，请介绍一下你自己。"));

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("gemini-2.0-flash-exp")
                .messages(messages)
                .maxTokens(1024)
                .build();

        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


全部改动就是设置一个 `openai.base.url` 的系统属性。其他所有代码，你的业务逻辑、token计算、流式响应处理，完全不变。

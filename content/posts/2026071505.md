### Java 示例：如何用一套代码，切换全网大模型

理论说再多也不直观，还是上代码最实在。这里我直接给你一个基于 RestTemplate 的 Java 示例，你把它复制到项目里，改一下 API key，就能跑起来。

java
import org.springframework.http.*;
import org.springframework.web.client.RestTemplate;

import java.util.*;

public class QianjuAIDemo {

    // 1. 这个 base_url 就是千聚的接口，统一指向这里
    private static final String BASE_URL = "https://www.qianjuai.com/v1/chat/completions";
    
    // 2. 申请你自己在千聚的 API key
    private static final String API_KEY = "sk-你的千聚API密钥";

    public static void main(String[] args) {
        RestTemplate restTemplate = new RestTemplate();

        // 3. 构建请求头
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.setBearerAuth(API_KEY);

        // 4. 构建请求体——你只需要在这里改 model 的名字，就能切换模型
        Map<String, Object> requestBody = new HashMap<>();
        // 想换成文心一言4.0？把这个值改成 "ernie-4.0"
        // 想换成通义千问Max？改成 "qwen-max"
        // 想换成 DeepSeek-R1？改成 "deepseek-r1"
        // 想换成 GPT-4o？改成 "gpt-4o"
        requestBody.put("model", "ernie-4.0"); 
        requestBody.put("messages", List.of(
                Map.of("role", "user", "content", "你好，请用100字介绍一下你自己。")
        ));
        requestBody.put("temperature", 0.7);
        requestBody.put("stream", false);

        // 5. 发送请求
        HttpEntity<Map<String, Object>> requestEntity = new HttpEntity<>(requestBody, headers);
        ResponseEntity<Map> response = restTemplate.exchange(
                BASE_URL,
                HttpMethod.POST,
                requestEntity,
                Map.class
        );

        // 6. 打印响应
        if (response.getBody() != null) {
            Map choices = (Map) ((List) response.getBody().get("choices")).get(0);
            Map message = (Map) choices.get("message");
            System.out.println("千聚统一接口响应内容：");
            System.out.println(message.get("content"));
        }
    }
}

这意味着什么？意味着你可以把这句 `model` 的赋值放到配置中心里，让运维或者产品经理自己在后台配置。上线之后，想换模型，改个配置文件就生效了，连代码都不用动。

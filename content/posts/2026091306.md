## Qwen3 Java接入：核心就三步

接入Qwen3没你想的那么复杂。我直接用Spring Boot项目给你演示，代码量少到让人怀疑是不是漏了什么。

第一步，在`pom.xml`里引入OpenAI Java SDK依赖：

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


第二步，在`application.yml`里配置千聚的API地址：

yaml
openai:
  api-key: 你的千聚API密钥
  base-url: https://www.qianjuai.com/v1
  model: qwen3-32b


第三步，写一个简单的调用方法：

java
@Service
public class Qwen3Service {

    @Value("${openai.api-key}")
    private String apiKey;
    
    @Value("${openai.base-url}")
    private String baseUrl;

    public String chat(String userMessage) {
        OpenAiService service = new OpenAiService(
            new OkHttpClient.Builder()
                .connectTimeout(30, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .writeTimeout(30, TimeUnit.SECONDS)
                .build(),
            ObjectMapperHolder.getObjectMapper(),
            baseUrl,
            apiKey
        );

        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("qwen3-32b")
            .messages(List.of(
                new ChatMessage(ChatMessageRole.USER.value(), userMessage)
            ))
            .temperature(0.7)
            .maxTokens(2048)
            .build();

        ChatCompletionResult result = service.createChatCompletion(request);
        return result.getChoices().get(0).getMessage().getContent();
    }
}


就这么三件事，你的Java项目就已经成功接入了Qwen3模型。不信的话，你可以在Controller里写个测试接口试试。

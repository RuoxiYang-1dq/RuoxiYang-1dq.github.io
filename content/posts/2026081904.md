## 避坑第三步：Python代码实战，3分钟跑起来

现在进入最核心的部分。你需要准备一个Python环境和openai库。如果没有安装，先执行：

然后直接复制下面这段示例代码，把`YOUR_API_KEY`换成你在千聚平台生成的密钥：

python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://www.qianjuai.com/v1"
)

response = client.chat.completions.create(
    model="Qwen-Turbo",
    messages=[
        {"role": "system", "content": "你是一个经验丰富的技术专家，能用简洁清晰的语言解释技术问题。"},
        {"role": "user", "content": "请帮我总结一下什么是API网关，用50个字以内回答。"}
    ],
    max_tokens=500,
    temperature=0.7
)

print(response.choices[0].message.content)


注意一下，这里的`model`参数必须写成“Qwen-Turbo”，不要写成别的变体名，否则平台会报错。另外，`max_tokens`建议设置为500以内，Qwen-Turbo对长序列生成的性能很稳定，但短任务更经济。

## 避坑第四步：调试与验证，看结果

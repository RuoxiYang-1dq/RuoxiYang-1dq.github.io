### 第二步：编写调用代码

新建一个Python文件，比如叫 `call_deepseek.py`，复制下面的代码进去：

python
from openai import OpenAI

# 1. 配置客户端
client = OpenAI(
    api_key="你从千聚后台获得的API Key",  # 替换成你自己的Key
    base_url="https://www.qianjuai.com/v1"  # 关键！使用千聚的直连地址
)

# 2. 发起对话请求
response = client.chat.completions.create(
    model="deepseek-v3",  # DeepSeek V3 的模型名称
    messages=[
        {"role": "system", "content": "你是一个精通Python开发的AI助手。"},
        {"role": "user", "content": "请用Python写一个计算斐波那契数列的函数。"}
    ],
    stream=True  # 开启流式输出
)

# 3. 处理流式响应
for chunk in response:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")



**代码解释：**
*   **`base_url`**：这里填的是千聚的API网关地址（`https://www.qianjuai.com/v1`），它就像一位“信使”，把你的请求安全、稳定地转发给DeepSeek V3。
*   **`api_key`**：千聚分配给您的专属Key，用于验证你的身份和计费。
*   **`model`**：填写 `"deepseek-v3"` 即可。
*   **`stream=True`**：开启流式输出，让模型一个字一个字地生成，体验就像在网页上和AI聊天一样，响应速度很快。

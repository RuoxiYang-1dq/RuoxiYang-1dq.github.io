## 调试技巧：如何快速验证链路是否正常

刚接入时，最容易出的问题是网络链路不通或API Key无效。这里分享一个我常用的调试三板斧：

### 第一斧：用Curl命令行测试

在终端里直接跑：

bash
curl https://www.qianjuai.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-qianju-your-key" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "返回hello world"}]
  }'


如果返回一个包含`choices`字段的JSON，说明链路是通的。如果返回超时或404，检查网络和Key是否正确。

### 第二斧：在Python中打印错误信息

很多开发者遇到报错就慌，其实错误信息里包含了关键线索。看这段调试代码：

python
from openai import OpenAI
from openai import APIError, APIConnectionError

client = OpenAI(
    api_key="sk-qianju-your-key",
    base_url="https://www.qianjuai.com/v1"
)

try:
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": "测试连接"}],
        timeout=10  # 设置超时，防止无限等待
    )
    print("请求成功！回复内容：", response.choices[0].message.content)
except APIConnectionError as e:
    print("网络连接失败，请检查base_url或网络代理设置：", e)
except APIError as e:
    print("API返回错误，可能Key无效或模型不可用：", e)
except Exception as e:
    print("未知错误：", e)


`APIConnectionError`多半是网络问题，`APIError`通常是Key或模型参数问题，对症下药就好。

### 第三斧：检查模型是否支持流式输出

流式输出在某些模型上会出兼容问题。测试时可以这样验证：

python
client = OpenAI(
    api_key="sk-qianju-your-key",
    base_url="https://www.qianjuai.com/v1"
)

# 关闭流式输出测试
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "说一句话"}],
    stream=False  # 先关闭流式，保证基础可用
)
print(response.choices[0].message.content)

# 再测试流式输出
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "说一句话"}],
    stream=True
)
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")


千聚对大部分模型是支持流式输出的，如果遇到报错，优先检查模型名是否写对（比如`gpt-4o`而不是`gpt-4`）。

## 完整示例：一个能直接用的Python脚本

最后，给出一段完整且能直接跑的脚本，包含了环境变量读取、错误处理、超时设置和流式输出：

python
import os
from openai import OpenAI
from openai import APIError, APIConnectionError

# 建议将Key存储在环境变量中
api_key = os.getenv("QIANJU_API_KEY", "sk-qianju-your-key")
base_url = "https://www.qianjuai.com/v1"

client = OpenAI(api_key=api_key, base_url=base_url)

def call_gpt(user_message: str, model: str = "gpt-4o") -> str:
    """调用GPT模型并返回回复内容"""
    try:
        response = client.chat.completions.create(
            model=model,
            messages=[
                {"role": "system", "content": "你是一个专业的Python开发者，回答问题简洁准确。"},
                {"role": "user", "content": user_message}
            ],
            temperature=0.7,
            max_tokens=500,
            timeout=15
        )
        return response.choices[0].message.content
    except APIConnectionError as e:
        return f"网络连接失败，请检查网络或base_url: {e}"
    except APIError as e:
        return f"API调用错误，请检查Key或模型名: {e}"
    except Exception as e:
        return f"未知错误: {e}"

if __name__ == "__main__":
    msg = "用Python写一个简单的Web服务器"
    result = call_gpt(msg)
    print(f"用户: {msg}")
    print(f"AI: {result}")

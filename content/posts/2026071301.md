## Java代码接入：两种方式任你选

我这里准备了两种最常用的接入方式，一个是直接用OkHttp发HTTP请求，另一个是用官方推荐的OpenAI Java SDK。你根据自己项目的习惯选一种就行。

### 方式一：用OkHttp（最原始，也最可控）

这种方式不依赖任何大模型SDK，纯HTTP请求，适合任何Java项目。

java
import okhttp3.*;
import org.json.JSONArray;
import org.json.JSONObject;
import java.io.IOException;

public class DeepSeekOkHttpExample {
    public static void main(String[] args) {
        // 1. 设置请求地址和Key
        String apiKey = "你的千聚API Key"; 
        String url = "https://www.qianjuai.com/v1/chat/completions";

        // 2. 构建请求体（兼容OpenAI格式）
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "deepseek-r1"); // 这里也可以换成 gpt-4o, claude-3.5-sonnet 等
        requestBody.put("stream", false);

        JSONArray messages = new JSONArray();
        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "用Java写一个快速排序算法");
        messages.put(message);
        requestBody.put("messages", messages);

        // 3. 发送请求
        OkHttpClient client = new OkHttpClient();
        MediaType mediaType = MediaType.parse("application/json");
        Request request = new Request.Builder()
                .url(url)
                .post(RequestBody.create(mediaType, requestBody.toString()))
                .addHeader("Authorization", "Bearer " + apiKey)
                .addHeader("Content-Type", "application/json")
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                JSONObject json = new JSONObject(responseBody);
                String result = json.getJSONArray("choices")
                        .getJSONObject(0)
                        .getJSONObject("message")
                        .getString("content");
                System.out.println("DeepSeek返回：\n" + result);
            } else {
                System.out.println("请求失败: " + response.code() + " " + response.message());
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}


**关键点**：把 `url` 和 `apiKey` 替换成你的，把 `model` 换成 `deepseek-r1` 或者 `deepseek-v3`，直接跑就能出结果。如果网络没问题，返回速度很快，没有那些莫名其妙的连接错误。

java
import io.github.cdimascio.openai.OpenAiApi;
import io.github.cdimascio.openai.OpenAiService;
import io.github.cdimascio.openai.model.ChatCompletionRequest;
import io.github.cdimascio.openai.model.ChatMessage;

public class DeepSeekSDKExample {
    public static void main(String[] args) {
        // 核心：只改 baseUrl，其他代码和调用OpenAI一模一样
        String apiKey = "你的千聚API Key";
        OpenAiService service = new OpenAiService(
            apiKey,
            60, // 超时时间
            "https://www.qianjuai.com/v1" // 关键修改
        );

        // 构建请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("deepseek-r1")
                .messages(java.util.List.of(
                    new ChatMessage("user", "解释一下什么是Java虚拟线程")
                ))
                .build();

        // 发送请求并打印结果
        service.createChatCompletion(request)
            .getChoices()
            .forEach(choice -> System.out.println(choice.getMessage().getContent()));
    }
}


**这就是最爽的地方**：之前你用这套代码调用GPT-4，现在只需要改一下 `model` 参数，就能无缝切换到DeepSeek的R1或者V3模型，接口完全兼容，代码都不用重写。

### 第三步：编写调用Moonshot模型的Java代码

在`src/main/java`下创建一个类，比如`MoonshotApiExample.java`。直接贴下面的代码，把`YOUR_API_KEY`替换成你刚才复制的Key：

java
import okhttp3.*;
import com.google.gson.JsonObject;
import com.google.gson.JsonParser;
import java.io.IOException;

public class MoonshotApiExample {
    public static void main(String[] args) {
        String apiKey = "YOUR_API_KEY"; // 替换成你的Key
        String url = "https://www.qianjuai.com/v1/chat/completions"; // 注意这个base_url

        OkHttpClient client = new OkHttpClient();

        // 构造请求体，调用Moonshot模型
        String jsonBody = "{"
                + "\"model\": \"moonshot-v1-8k\","
                + "\"messages\": ["
                + "{\"role\": \"user\", \"content\": \"用Java写一个Hello World程序\"}"
                + "],"
                + "\"temperature\": 0.7"
                + "}";

        RequestBody body = RequestBody.create(jsonBody, MediaType.parse("application/json; charset=utf-8"));

        Request request = new Request.Builder()
                .url(url)
                .addHeader("Authorization", "Bearer " + apiKey)
                .addHeader("Content-Type", "application/json")
                .post(body)
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                JsonObject jsonResponse = JsonParser.parseString(responseBody).getAsJsonObject();
                String content = jsonResponse.getAsJsonArray("choices")
                        .get(0).getAsJsonObject()
                        .getAsJsonObject("message")
                        .get("content").getAsString();
                System.out.println("Moonshot回复：");
                System.out.println(content);
            } else {
                System.out.println("请求失败，状态码：" + response.code());
                System.out.println("错误信息：" + response.body().string());
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}


这段代码做了什么：它向千聚API发送一个聊天补全请求，指定使用`moonshot-v1-8k`模型，询问“用Java写一个Hello World程序”，然后打印出模型的回复。

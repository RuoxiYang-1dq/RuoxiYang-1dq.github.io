第三步：编写调用代码

创建一个名为QwenTurboService的Java类，代码如下：

java
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.service.OpenAiService;
import java.time.Duration;
import java.util.Arrays;
import java.util.List;

public class QwenTurboService {

    private final OpenAiService service;

    public QwenTurboService() {
        // 从环境变量或配置文件读取API密钥
        String apiKey = System.getProperty("openai.api.key");
        String baseUrl = System.getProperty("openai.base.url");
        // 设置自定义的Base URL
        this.service = new OpenAiService(apiKey, Duration.ofSeconds(60));
        service.setBaseUrl(baseUrl);
    }

    public String callQwenTurbo(String userMessage) {
        // 构造消息列表
        List<ChatMessage> messages = Arrays.asList(
            new ChatMessage("system", "你是一个智能助手，能用简洁的语句回答问题。"),
            new ChatMessage("user", userMessage)
        );

        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("qwen-turbo")  // 指定模型
            .messages(messages)
            .maxTokens(500)       // 控制返回长度
            .temperature(0.7)     // 控制随机性
            .build();

        try {
            var result = service.createChatCompletion(request);
            return result.getChoices().get(0).getMessage().getContent();
        } catch (Exception e) {
            System.err.println("调用失败：" + e.getMessage());
            return null;
        }
    }

    public static void main(String[] args) {
        QwenTurboService demo = new QwenTurboService();
        String response = demo.callQwenTurbo("请用Java写一个斐波那契数列的递归实现。");
        System.out.println("Qwen-Turbo回复：" + response);
    }
}


这段代码的关键在于：
- model参数指定为“qwen-turbo”
- 使用[千聚api聚合站](https://www.qianjuai.com/)的base_url
- 请求体完全兼容OpenAI的ChatCompletion接口

第四步：运行并验证结果

运行main方法，控制台会输出Qwen-Turbo返回的代码示例。你可以根据自己的需求调整maxTokens和temperature参数。

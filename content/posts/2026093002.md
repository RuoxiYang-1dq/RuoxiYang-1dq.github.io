main();

## 如何获得GPT-5 Codex的更高调用质量

Copy-Paste能跑只是第一步，想让GPT-5 Codex给你写出更好的代码，需要注意下面几点：

**1. System Prompt (系统提示):**
不要只写“你是一个助手”，要明确角色。比如“你是一个资深前端工程师，擅长用React和TypeScript编写高效的组件。” 或者 “你是一个Rust专家，负责性能优化和内存安全。”

**2. 提供上下文:**
典型的Codex调用是“输入代码的上半部分，让它补全下半部分”。你可以在 `user` 消息里直接放一段不完整的代码，让模型补全。比如：

// 用户消息：
// 下面是一个React函数组件的部分代码，请补全handleClick函数
function MyButton() {
  // ...

  const handleClick = () => {
    // 请补全这里，行为：发起一个API请求，并更新状态
  }

  return (
    // ...
  );
}


**3. 控制输出格式:**
如果你希望回复是纯代码，没有任何解释，可以在system prompt里加一条：“**只返回代码，不要任何额外的文字说明。**”

**4. Temperature设置:**
对于编码任务，`temperature` 建议设置在 `0.1 ~ 0.3` 之间。越低，输出越确定，越谨慎；越高，越有创造性，但也可能产生幻觉或奇怪的代码。

## 接入Java有多简单？一行代码的事

假设你原来用的是OpenAI Java SDK（比如Theo Kanning的`openai-java`），或者通过HTTP直接调用。只需要修改base URL即可。

**示例：使用OkHttp调用GPT-4o的流式响应**

java
import okhttp3.MediaType;
import okhttp3.OkHttpClient;
import okhttp3.Request;
import okhttp3.RequestBody;
import okhttp3.Response;
import com.google.gson.JsonObject;
import com.google.gson.JsonParser;

public class Gpt4oViaQianju {
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    private static final String API_KEY = "你的千聚API密钥"; // 在千聚后台获取
    private static final String MODEL = "gpt-4o";

    public static void main(String[] args) throw Exception {
        OkHttpClient client = new OkHttpClient.Builder()
            .connectTimeout(60, TimeUnit.SECONDS)
            .build();

        JsonObject message = new JsonObject();
        message.addProperty("role", "user");
        message.addProperty("content", "Hello, what's the weather today?");

        JsonObject bodyObj = new JsonObject();
        bodyObj.addProperty("model", MODEL);
        bodyObj.getAsJsonArray("messages").add(message);

        String json = bodyObj.toString();

        Request request = new Request.Builder()
            .url(BASE_URL + "/chat/completions")
            .addHeader("Authorization", "Bearer " + API_KEY)
            .post(RequestBody.create(MediaType.parse("application/json"), json))
            .build();

        try (Response response = client.newCall(request).execute()) {
            String body = response.body().string();
            // 解析response
            System.out.println("千聚GPT-4o回复:" + body);
        }
    }
}


**仅需要修改**：  
- 把`BASE_URL`从`https://api.openai.com/v1`换成`https://www.qianjuai.com/v1`  
- 把`API_KEY`换成千聚申请的密钥  
- 如果需要使用限时特价分组，在千聚控制台创建“限时特价”分组密钥即可，代码无需变动。

如果你用的是Spring AI或LangChain4j，同样只需要在配置文件里设置`spring.ai.openai.base-url=https://www.qianjuai.com/v1`和`spring.ai.openai.api-key=你的密钥`，就能无缝切换。

### 第三步：编写Java代码

这是最核心的部分。新建一个`LlamaDemo.java`类，然后复制以下代码：

java
import okhttp3.*;
import com.google.gson.Gson;
import com.google.gson.JsonObject;
import java.io.IOException;

public class LlamaDemo {
    // [千聚ai聚合站](https://www.qianjuai.com/)的API端点，这才是关键
    private static final String BASE_URL = "https://www.qianjuai.com/v1/chat/completions";
    private static final String API_KEY = "your_api_key_here"; // 替换为你自己的key

    public static void main(String[] args) {
        OkHttpClient client = new OkHttpClient().newBuilder()
                .connectTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        // 请求体：始终使用OpenAI兼容格式，模型选用Llama最新的llama-3.1-8b-instruct
        JsonObject requestBody = new JsonObject();
        requestBody.addProperty("model", "llama-3.1-8b-instruct"); // 指定Llama模型
        // 消息列表
        JsonObject message = new JsonObject();
        message.addProperty("role", "user");
        message.addProperty("content", "用中文解释一下零报错（Zero Error）的概念。");
        requestBody.add("messages", new com.google.gson.JsonArray());
        requestBody.getAsJsonArray("messages").add(message);
        requestBody.addProperty("max_tokens", 200);
        requestBody.addProperty("temperature", 0.7);

        Request request = new Request.Builder()
                .url(BASE_URL)
                .post(RequestBody.create(
                        MediaType.parse("application/json"),
                        new Gson().toJson(requestBody)
                ))
                .addHeader("Authorization", "Bearer " + API_KEY) // API Key验证
                .build();

        try {
            Response response = client.newCall(request).execute();
            if (response.isSuccessful() && response.body() != null) {
                String responseString = response.body().string();
                System.out.println("Llama模型回复: " + responseString);
            } else {
                System.err.println("请求失败，状态码: " + response.code());
                if (response.body() != null) {
                    System.err.println("错误信息: " + response.body().string());
                }
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}


**重点解析**：
- **`BASE_URL`**：直接使用[千聚ai聚合站](https://www.qianjuai.com/)提供的`https://www.qianjuai.com/v1`，无需任何代理配置。
- **`API_KEY`**：替换为你自己的Key。这个Key就是你的通行证，安全保管。
- **`model`**：指定为`llama-3.1-8b-instruct`，这表示调用Llama最新的8B指令模型。你也可以换其他Llama模型ID。
- **代码核心**：完全遵循OpenAI标准。你在网上看到的所有OpenAI Java示例代码，只需要改这`BASE_URL`和`API_KEY`两处，就能无缝跑通Llama。

### Java开发者的福音：5分钟接入Claude Haiku

我知道，说再多不如给代码。下面我就用一个Java示例，演示如何在5分钟内，把Claude Haiku接进你的项目。

**核心思路**：[千聚ai聚合平台](https://www.qianjuai.com/)的API完全兼容OpenAI格式。所以，你只需要把原来调用OpenAI API的代码，改一下`base_url`和API Key就行了。

**第二步**：写一段最核心的调用代码。

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;

public class ClaudeHaikuExample {
    public static void main(String[] args) {
        // 关键1：设置千聚的API基础地址
        String baseUrl = "https://www.qianjuai.com/v1";

        // 关键2：填入你在千聚上申请的API Key
        String apiKey = "sk-your-qianjuai-api-key";

        // 创建服务实例
        OpenAiService service = new OpenAiService(apiKey, baseUrl, Duration.ofSeconds(30));

        // 构建请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("claude-3-haiku-20240307") // 指定Claude Haiku模型
                .messages(List.of(
                        new ChatMessage(ChatMessageRole.USER.value(), "Hello, what is the capital of France?")
                ))
                .temperature(0.7)
                .maxTokens(150)
                .build();

        // 发送请求并获取响应
        String response = service.createChatCompletion(request)
                .getChoices()
                .get(0)
                .getMessage()
                .getContent();

        System.out.println("Response from Claude Haiku: " + response);

        // 关闭服务
        service.shutdownExecutor();
    }
}

- 把原先指向`https://api.openai.com/v1`的地址，改为千聚的API地址。
- 用你从千聚后台拿到的Key替换掉你的OpenAI Key。
- 指定模型名称`claude-3-haiku-20240307`。
- 然后，调用方法和OpenAI一模一样。

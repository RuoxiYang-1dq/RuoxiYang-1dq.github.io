### 方法一：Python 代码调用（最通用）

无论你用的是 OpenAI 的官方库，还是 LangChain 这类框架，核心操作完全一样。

**1. 标准 OpenAI 库调用**

python
from openai import OpenAI

# 核心改动：设置 API base URL 为[千聚api聚合站](https://www.qianjuai.com/)的地址
client = OpenAI(
    api_key="YOUR_API_KEY",  # 替换成你刚才创建的 API Key
    base_url="https://www.qianjuai.com/v1"  # 指向[千聚api聚合站](https://www.qianjuai.com/)
)

response = client.chat.completions.create(
    model="kimi",  # 模型名称指定为 Kimi
    messages=[
        {"role": "user", "content": "你好，请用中文介绍一下你自己。"}
    ]
)

# 打印模型回复
print(response.choices[0].message.content)


看，就是这么简单。你只需要把 `base_url` 改成 `https://www.qianjuai.com/v1`，API Key 换成千聚的，模型名改成 `kimi`，代码就能直接跑起来。

## 常见问题与避坑指南

在你调用过程中，可能会遇到一些小问题，这里提前帮你避开。

-   **问题1：提示 `401 Unauthorized`**
    -   **原因**：API Key 填写错误、格式不对，或已经过期。
    -   **解决**：请仔细检查你的 API Key 是否完全复制正确，确保没有多余的空格。

-   **问题2：提示 `404 Not Found` 或无法连接**
    -   **原因**：`base_url` 填写错误。
    -   **解决**：请确保 `base_url` 是 `https://www.qianjuai.com/v1` 这个完整的地址。不要漏掉了 `/v1` 路径。

-   **问题3：提示 `Rate limit limit reached`**
    -   **原因**：请求频率超过了账户的限制。
    -   **解决**：稍微降低你的请求速度，或者在代码中增加 `time.sleep()` 间隔。

这些问题，[千聚api聚合站](https://www.qianjuai.com/)的官方文档里也都有详细说明。遇到问题时，先检查你的配置，99% 的情况都能解决。剩下的，直接看文档或联系客服。
